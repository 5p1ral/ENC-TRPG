【转载】KP夜话:如何制裁刁民？

发表于 2020-06-16 更新于 2020-06-30 分类于 跑团杂谈
本文字数： 2.9k 阅读时长 ≈ 3 分钟

原作者：问号
原文地址：https://www.bilibili.com/read/cv6436432

这篇文稿算是和别人聊的兴起，临时想到写这么篇文章出来算了，大家看时间也就发现，我确实是脑子一抽就开始写，写完了就直接发，没有校验没有精修（本来也没有）。因此也不敢当什么教程或者攻略，只能算是个人的一点粗略的经验分享，也希望借此抛砖引玉能和大家聊聊。

首先，作为一个手法老练的GM，我其实是非常鄙夷通过比如说“天上掉下一个什么东西。”或者“突然如何如何，于是乎你死了。”这类突发且无厘头的事情来作为我与刁民之间交锋的结局。因为我觉得这么做实在是太low了，这就相当于在群里和别人激情对线对不过然后直接把对方禁言了一样，这简直就是一种耻辱。

所以在我看来，针对刁民的手法不应当一概而论，而是要根据对方的特点和行为，有针对有区分的去一一瓦解。

第一类刁民：喜欢超游的

严格来说超游似乎并不算太过严重的刁民，这类玩家或许是想要让自己的角色更加光彩，或许是想要找到更多存在感，又或者对方是确确实实的迫切希望能迎来好的结局。

对付这种玩家往往我的手段会比较温和，通过一点点的诱导，比如一个让对方拿捏不准的暗骰，再加上一点点的暗示，比如一句非常富有引导意味的语言“那里什么也没有，没有！”，再加上对故事的一点点改动。

最后在对方大动干戈的探索调查之后，换来一个玩笑一般的结局（“啤凉钟吊”）。大家啼笑皆非一下，点到即止。

提醒到了即可，没必要你死我活。

第二类刁民：喜欢整活的

整活类型的刁民，比如“火车王”或者是“移动军火库”又或者是“我要潜行偷窥女npc洗澡……我要潜行偷窥男npc洗澡！”甚至是“18岁精通三国语言擅长六种学科能够徒手制造炸弹但是在危机面前不动声色只想安安静静当个普通人的”（说的是谁看到了请自行面壁），和整活玩家还是有些不同的。

这个不同主要是要看团中搞事的玩家有多少，是所有人都在谨慎调查只有一个玩家开火车？还是整个团队跟吃了炫迈一样冲冲冲根本停不下来？一般对于后者而言，个人的行为倒也算不上刁民，只能说整个团队的团风比较魔性，比较狂欢。所以这里我们只认为前者算是刁民。

很多时候从情节上看，让这个喜欢整点活的玩家因为自己的胡乱操作而送命是很符合逻辑的处理方式，甚至更进一步这可能会因为他一个人的奇葩操作让整个队伍一波血崩。但我还是觉得这么做似乎不妥——要么太过笼统，要么太过苛刻。

通常我更喜欢用一种折中的方式，让全队为了他的莽撞而陷入危险之中，但仅仅只是危险。通过设置一些惊险的场景来迫使整个团队为了生存挣扎一段时间，最后通过一种“幸运”（其实是精心策划好的）的办法最后脱离险境。

想必此时他也能明白自己的奇葩操作将会为全队带来什么样的后果，接下来的故事他便能谨慎许多。

让对方明白自己做是不好的，也明白自己做为什么不好（一人行为全队背锅），最后还不会造成太大的麻烦来损害其他玩家的体验，一举两得。

第三类刁民：恶意破坏他人游戏体验的

这是一个比较严肃的话题，在排除了秘密团、二五仔等所有玩家都早有准备并且认同的前提下，这种事情发生几乎可以从“麻烦”的层面上升到了“事故”的级别了。

举个例子，当团中你的一群PL千辛万苦从危难中拯救了一名NPC的命运，但其中的一位二五仔却决定要将这个角色出卖掉以换取利益。这不仅代表了其他所有PL前期的努力共同付诸东流，更能得以预见的是这势必会引起玩家之间的矛盾，现实中的。

我们可以很轻易地发现在这个矛盾之中似乎每个玩家地行为都是“正确”的，他们能为自己的理由找出无数的基于人物设定、基于逻辑、基于剧情必须甚至是基于所谓的“理性”的选择。但这依旧是一场矛盾。

凭心而论，这种矛盾的发生背后其实也包含了一部分GM的责任——GM作为游戏的管理者，实际上同样也肩负有游戏外团内的管理责任，从招募玩家的一刻开始GM就应该尽可能地保证自己地玩家们之间不会产生这种矛盾——这不是游戏中角色的冲突，而是游戏外玩家们之间理念与观点的冲突——正如为什么D&D中喜欢将玩家按照阵容九宫格粗略划分一样。

众所周知，观念的冲突只会引发纠纷，难以通过语言达成共识。

虽然发生的概率不高，但在当下网团的环境中，大量充斥着需要临时拼凑才能组成的跑团团队而言，KP的确难以保证所有的玩家的观念不会相互矛盾。更无法避免本身就是别有用心的人混入了团中。

那么当这种矛盾真实发生的时候，作为GM，我觉得我们是时候让我们的故事停下来了。这个时候就不要幻想继续维持故事的好与坏的时候，而是应该站出来，以管理员的方式去维持玩家的这个团队——往往这种矛盾一旦处理不好，整个团将会瞬间分崩离析。

究竟该怎么做？

首先，我或许会打断当下的游戏进程，转而严肃的询问玩家。

“你这么做会引发所有人对你的不满，你确定吗？”

如果对方仍然坚持他的行为，那么接下来，让对方阐述他的理由，让被他的行为损害到的玩家去做出决定，接受或者不接受。这个过程中我们可以好言相劝，也可以沉默不发，任由玩家自己决定。

如果在此之后我们仍然无法开解这个矛盾的话，最不幸的一幕便发生了，我们将不得不选择矛盾中的一方，让他从这个团队中离开，接下来或许GM会操纵他的角色，又或许通过种种手段让这个角色离开剧本的舞台。

但在决定谁要离开之后，我们还应该明确的，毫不隐讳的告诉这位玩家，他是为什么离开的——“也许你没有错，但你破坏了其他人的游戏体验，所以你不适合这个团队。”

身为KP，作为团队的管理者，除了学会如何讲好一个故事意外，我们有，也应当有引导我们的玩家能够学会如何做好一个玩家的职责。当矛盾发生的时候我们不光要解决矛盾，也应该去试着解决矛盾的本质，去告诉玩家他哪里做的不好，为什么？

说到这里，让我们重新回顾所谓“刁民”这一行为的本质。什么是刁民？是超游行为本身吗？是不好好动脑子吗？是不扮演不RP吗？是无脑踢门开怪火车王吗？

都不是。判断是不是刁民的标准永远只有一个——

玩家的行为是否损害了跑团中玩家这一群体中其他成员的共同游戏体验，包括其他PL，也包括KP。

COC是跑团，是桌面游戏，是一种社交性质的游戏，是KP和PL一起讲故事的游戏。但这本质，还是一个游戏，甚至于所谓的COB也好其本质同样也是一个游戏。是游戏，就是要给玩家带来快乐，带来感悟的，而有的玩家破坏了其他玩家的游戏体验，这就是刁民。

刁民与否，从来都没有一个明确的界限，而是根据团队的不同而有着不同的范畴，同样的行为在有的团里是精彩的，在有的团里便是令人厌恶的。我们不能也不该要求所有的团都是一个模样。但我们更应该明白，跑团无论如何都要考虑到其他玩家的感受，正如KP要考虑PL的体验，PL也要体谅KP的辛苦一样。

身为KP，我们不光要明白这一点，要作到这一点，也同样应该包容我们的PL，引导我们的PL，告诉他们什么是对的，什么是错的。

惩罚刁民不是理由，而应当是引导PL明白究竟该如何去做的手段。

从团队踢出刁民也不是鄙夷与排斥，不是强权与暴力，更不应当成为一种高高在上的鄙夷与唾弃，不应成为一种正确的对不正确的批判。我们要明白，哪怕是将一些玩家踢出去，这也只是为了维护KP与PL共处的团队，而非为了正义。

和而不同，群而不党，敬而远之。

这便是我想说的。

聊的有些晚了，祝各位今夜做个好梦
